📘 Основы словообразования

🔹 Суффиксы существительных

-tion/-sion : educate → education
-ment : develop → development
-ness : happy → happiness
-ity/-ty : active → activity
-ence/-ance : different → difference
-er/-or : teach → teacher

🔸 Суффиксы прилагательных

-able/-ible : comfort → comfortable
-ful : wonder → wonderful
-less : hope → hopeless
-ive : create → creative
-al : nature → natural
-ous : danger → dangerous

🔹 Префиксы с отрицательным значением

un- : happy → unhappy
in-/im-/ir-/il- : possible → impossible
dis- : appear → disappear
non- : fiction → non-fiction
mis- : understand → misunderstand

🔸 Глагольные суффиксы и префиксы

-ize/-ise : modern → modernize
-en : wide → widen
re- : write → rewrite
over- : load → overload
